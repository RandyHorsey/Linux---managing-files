<h1>Linux - Manging Files</h1>

<h2>Description</h2>
Using Linux commands to modify a directory structure and the files it contains. Also, I used the nano text editor to add text to a file.
<br />

<h2>Linux commands</h2>

- <b>mkdir</b> 
- <b>rmdir</b>
- <b>touch</b>
- <b>rm</b>
- <b>mv</b>
- <b>cp</b>
- <b>nano</b>
